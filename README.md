# 🖤 Alpha Black - E-commerce

Projeto de e-commerce de roupas desenvolvido com foco em performance, design moderno e escalabilidade.

## 🚀 Tecnologias
- Next.js
- Node.js
- Express
- MongoDB

## 🔐 Funcionalidades
- Cadastro e login de usuários
- Listagem de produtos
- Carrinho de compras
- Sistema de autenticação JWT
